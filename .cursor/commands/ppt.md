# PPT / Keynote

把指定素材做成一套大会级、可直接放映的单文件 HTML 幻灯片。
遵循 .cursor/skills/ppt-builder/SKILL.md 的全部规范。

## 用法
/ppt {主题或素材}
例：/ppt 把 ./article.md 做成 15 页 Google I/O 风格分享，听众=开发者

## 执行步骤
1. 收集信息：主题、核心论点、听众、时长/页数、风格。缺则先问。
2. 先出分页大纲等用户确认（用户说"直接出"则跳过）。
3. 基于 SKILL.md 的骨架与组件生成单文件 HTML。
4. 保存为 {主题英文短名}-keynote.html 到项目根目录并提示打开预览。
5. 附每页演讲者备注。

## 默认参数
- 风格：Google I/O 四色　- 页数：12–16 页　- 语言：跟随素材
- vw/vh 自适应、深浅交替、键盘+点击翻页+进度条+全屏全部内置

## 红线
- 标题写结论不写名词；一页一观点；正文≤3行
- 不编造数据/案例，不确定处用 [TODO]
- 单文件、零 CDN、零依赖
- 正文不写仓库路径/文件名/文档链接，资源放备注或口播
- 不出现原文作者/公众号名；个人水印固定为 **云牧元宇宙**
