# 学习路线图

> 3 个月滚动计划 · 认知 → 工具 → Agent · 按自己节奏调整

---

## 阶段一 · 认知打底（第 1–4 周）

**目标**：建立「副驾」心智模型，能描述 AI 能/不能做什么。

| 周 | 主题 | 产出 |
|----|------|------|
| 1 | 能力边界、幻觉 | 更新 [cognition/00-ai-cognition-map.md](../cognition/00-ai-cognition-map.md) |
| 2 | Context 设计 | Essay：rules vs skills vs MCP |
| 3 | 播客输入 | 至少 2 条 [podcast/](../podcast/) 卡片 |
| 4 | 复盘 | `meta/` 下月度小结（可选） |

---

## 阶段二 · MCP 与工具（第 5–8 周）

**目标**：能独立做一个最小 MCP，并知道何时该用 workflow tool。

| 周 | 主题 | 产出 |
|----|------|------|
| 5 | MCP 协议与传输 | 跑通 [labs/mcp-hello/](../labs/mcp-hello/) |
| 6 | 工具设计 | 完成 [playbooks/mcp-design-checklist.md](../playbooks/mcp-design-checklist.md) 实践笔记 |
| 7 | 第二个 MCP | 新 lab 或扩展现有 lab |
| 8 | 安全与鉴权 | Essay 或 playbook 补充 |

---

## 阶段三 · Skills 与 Agent（第 9–12 周）

**目标**：Skill 可触发、Agent 工作流可复现。

| 周 | 主题 | 产出 |
|----|------|------|
| 9 | Cursor Skill | [labs/cursor-skill-template/](../labs/cursor-skill-template/) 定制一版 |
| 10 | Prompt 模式 | 充实 [labs/prompt-patterns/](../labs/prompt-patterns/) |
| 11 | 多步 Agent | `labs/agent-workflow-notes/`（可选新建） |
| 12 | 总复盘 | README「最新内容」更新 |

---

## 资源栈（默认）

| 工具 | 用途 |
|------|------|
| Cursor | IDE + Skills + Agent |
| MCP | 外部能力与数据 |
| Python FastMCP | 本地 MCP lab |
| GitHub Project | 选题与进度 |

---

## 度量（轻量）

- 每月：公开 essay ≥ 1 或 lab 迭代 ≥ 1
- 播客：有 takeaway 的卡片 ≥ 2/月
- 不追求数量；追求 **可复用 artifact**
